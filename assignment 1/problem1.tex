\section{Solution}

{\bf Base Case:}

$(1+x)^n \ge n \dot x$

{\bf Step Case:}

$(1+x)^{n+1} \ge (n+1) \dot x$ 

$(1+x)^n \dot (1+x) \ge x \dot (n+1) $ 

$(1+x)^n + x \dot (1+x)^n \ge x \dot n + x $ 

$ (1+x)^n \ge n \dot x $ ,Then we have:

$x \dot (1+x)^n \ge x $

$ (1+x)^n \ge 1 $
